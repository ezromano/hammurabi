The Hammurabi Project

Codifying U.S. law in machine-executable form

Download this project as a .zip file Download this project as a tar.gz file

The Hammurabi Project is an experiment in codifying U.S. law - such as the U.S. Code, the Code of Federal Regulations, state laws, etc. - as executable computer code.

The project's basic components are:

  1. A core library of C# classes and functions that facilitate the representation of legal knowledge.
  2. Legal rule files written in a domain specific language called Akkadian (.akk).
  3. A primitive compiler that converts the Akkadian files into C#.
  4. Unit tests for the core library and the legal rules.

More information can be found on the project wiki.
